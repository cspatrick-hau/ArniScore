Dataset Description

=> This dataset contains annotated images of Red Player, Blue Player, Red Stick and Blue Stick
=> 4 classes(Red Player, Blue Player, Red Stick and Blue Stick)

Multiple angles and practitioners

=> Bounding box coordinates for each classes

Annotation Format

=> The dataset includes annotations YOLO format with images, containing:
=> Bounding box coordinates (x, y, width, height)
